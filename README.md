# Neo Brutal

**Neo Brutal** is a bold, unapologetic Visual Studio Code theme featuring thick black borders, high contrast colors, and an intentionally raw aesthetic inspired by Neo-Brutalism design.

![Neo Brutal Banner](./images/banner.png)

## Features

- **Bold Design:** Thick 3-4px black borders on every UI element for maximum impact.
- **High Contrast:** Pure black text on bright backgrounds with vibrant accent colors.
- **Zero Subtlety:** No gradients, no soft shadowsâ€”just raw, honest design.
- **Aggressive Color Palette:** Hot pink, electric blue, lime green, and bold orange.
- **Intentionally Unpolished:** Embraces the "undesigned" aesthetic of Neo-Brutalism.
- **Maximum Readability:** High contrast ratios ensure code is crystal clear.
- **Bold Typography:** Keywords and functions styled with bold fonts for emphasis.

## Philosophy

Neo Brutal embraces the Neo-Brutalism design movement's core principles:

- **Honesty in Design:** No hiding behind subtle effectsâ€”every element is bold and visible
- **Functional Clarity:** Thick borders and high contrast make every UI element obvious
- **Rejection of Minimalism:** More is moreâ€”bold colors and strong typography
- **Raw Aesthetic:** Intentionally avoiding the "polished" look of traditional themes

## Installation

### Via VS Code Marketplace

1. Open **Visual Studio Code**.
2. Go to the **Extensions** view by clicking on the Extensions icon in the Activity Bar or pressing `Ctrl+Shift+X` (`Cmd+Shift+X` on macOS).
3. Search for **Neo Brutal**.
4. Click **Install**.
5. Once installed, click **Set Color Theme** to activate it.

### Manual Installation

1. Download the `.vsix` file from the [Releases page](https://github.com/sana2k/neo-brutal/releases).
2. Open VS Code.
3. Go to Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`).
4. Click the `...` menu at the top â†’ **Install from VSIX**.
5. Select the downloaded file.

## Recommended Settings

For the full Neo-Brutalism experience, add these to your `settings.json`:
```json
{
  "editor.fontFamily": "'Space Grotesk', 'Inter', 'Roboto', monospace",
  "editor.fontWeight": "600",
  "editor.fontSize": 14,
  "editor.lineHeight": 1.6,
  "editor.letterSpacing": 0.5,
  "workbench.colorTheme": "Neo Brutal"
}
```
